# XZ Inline Image Bot

Телеграм-бот на базе aiogram, который работает в inline-режиме и возвращает результаты поиска изображений через Bing.

> 🚀 Попробовать бота: [@Velikiarbyzz_bot](http://t.me/Velikiarbyzz_bot)

## Возможности

- Inline-поиск изображений по запросу пользователя.
- Асинхронная обработка и быстрый ответ через `InlineQuery`.
- Фильтрация ссылок и проверка, что ресурс действительно изображение.
- Простая конфигурация через `.env`.

## Плюсы бота

| Плюс | Почему это важно |
| --- | --- |
| Быстрый inline-ответ | Не нужно ждать — результаты появляются прямо в чате. |
| Асинхронная обработка | Бот не блокируется даже при больших запросах. |
| Проверка валидности изображений | Отсекает битые ссылки и не-изображения. |
| Минимальная настройка | Только токен и `ADMIN_ID` в `.env`. |
| Простая архитектура | Легко дополнять и поддерживать код. |

## Стек

- Python

## Быстрый старт

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` и добавьте токен бота и `ADMIN_ID` (user id администратора):

```env
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_id
```

3. Запустите бота:

```bash
python -m xz.app
```

## Использование

Откройте любой чат в Telegram и введите:

```
@имя_бота ваш_запрос
```

Бот вернет картинки в inline-режиме.

### Команда администратора

- `/stats` — показывает время работы бота и количество ошибок. Доступ только для пользователя с `ADMIN_ID`.

## Структура проекта

- `xz/app.py` — запуск бота и регистрация хендлеров.
- `xz/config.py` — загрузка переменных окружения.
- `xz/handlers/start.py` — команда `/start` с инструкциями и дисклеймером.
- `xz/handlers/inline.py` — обработчик inline-запросов.
- `xz/services/bing_images.py` — поиск и валидация изображений.

## Примечания

- Бот использует открытые результаты поиска изображений Bing.
- Ответственность за вводимые запросы и использование материалов лежит на пользователе.
